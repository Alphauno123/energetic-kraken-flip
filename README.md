# AI Product Photo Generator

This is a web application built with React, TypeScript, Tailwind CSS, and shadcn/ui, designed to help e-commerce businesses generate stunning AI product photos from a single uploaded image.

## Features

- **Upload Product Image:** Easily upload a single product image.
- **Select Styles:** Choose from various AI-generated photo styles (e.g., Studio Shot, Lifestyle Scene, Seasonal Theme).
- **Generate Photos:** Our AI transforms your product image into multiple high-quality visuals based on your selections.
- **Download & Share:** Download individual or all generated photos, or copy image links to share.
- **Responsive Design:** Optimized for both desktop and mobile experiences.
- **Dark Mode:** Supports light and dark themes.

## Getting Started

To run this project locally:

1.  **Install Dependencies:**
    ```bash
    npm install
    # or yarn install
    # or pnpm install
    ```
2.  **Start Development Server:**
    ```bash
    npm run dev
    # or yarn dev
    # or pnpm dev
    ```
    The application will be available at `http://localhost:8080`.

## Project Structure

- `src/pages`: Contains the main application pages.
- `src/components`: Reusable UI components.
- `src/components/ui`: shadcn/ui components.
- `src/hooks`: Custom React hooks for encapsulating logic.
- `src/utils`: Utility functions and constants.
- `src/globals.css`: Global Tailwind CSS styles.
- `tailwind.config.ts`: Tailwind CSS configuration.
